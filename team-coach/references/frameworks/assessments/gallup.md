# Gallup CliftonStrengths — Referência de Interpretação

## Os 4 Domínios

### Executing — Fazem acontecer

| Força | Essência | Risco de excesso | Implicação para PM |
|---|---|---|---|
| **Achiever** | Drive constante para completar e produzir | Workaholic; pode pressionar o time sem perceber | Excelente em delivery — calibrar para não pular etapas de discovery |
| **Arranger** | Organiza pessoas e recursos para máxima eficiência | Pode reorganizar demais, gerando instabilidade | Ótimo para coordenação cross-funcional |
| **Belief** | Valores fortes guiam cada decisão | Pode ser inflexível quando os valores são testados | Ancora decisões em propósito — cuidado com dogmatismo |
| **Consistency** | Trata todos com equidade e segue processos claros | Pode resistir a exceções necessárias | Bom para criar processos confiáveis — calibrar flexibilidade |
| **Deliberative** | Identifica riscos antes de agir | Pode ser percebido como pessimista ou bloqueante | Excelente para pre-mortem e gestão de risco |
| **Discipline** | Cria estrutura, rotinas e ordem | Pode ser rígido em contextos que exigem agilidade | Bom para rituais de time — calibrar para aceitar o imperfeito |
| **Focus** | Mantém o objetivo principal mesmo com distrações | Pode ignorar oportunidades laterais relevantes | Excelente para manter produto no caminho — cuidado com bloqueio de pivôs |
| **Responsibility** | Assume compromissos e os honra | Pode assumir trabalho demais, sem delegar | Confiável para o time — trabalhar delegação e o "não" |
| **Restorative** | Identifica e resolve problemas com energia | Pode focar demais no que está errado, ignorando o que funciona | Bom para debug de produto e retrospectivas |

### Influencing — Lideram, falam, se fazem ouvir

| Força | Essência | Risco de excesso | Implicação para PM |
|---|---|---|---|
| **Activator** | Transforma pensamento em ação agora | Pode agir antes de validar premissas | Excelente para destravar times — calibrar com discovery |
| **Command** | Presença e clareza em situações ambíguas | Pode ser percebido como dominador | Bom para decisões difíceis — trabalhar escuta ativa |
| **Communication** | Transforma ideias em histórias que engajam | Pode priorizar apresentação sobre substância | Excelente para storytelling de produto |
| **Competition** | Motivado por comparação e vencer | Pode criar ambiente competitivo dentro do time | Canalizar para benchmarks externos, não dinâmicas internas |
| **Maximizer** | Foca no que está indo bem para elevar ainda mais | Pode negligenciar o que está abaixo da média | Excelente para identificar oportunidades de amplificação |
| **Self-Assurance** | Confiança interna para tomar riscos | Pode ser impermeável a feedback | Bom para decisões sob incerteza — trabalhar abertura |
| **Significance** | Quer ter impacto reconhecido | Pode tomar decisões por visibilidade, não impacto | Canalizar para impacto real — não para protagonismo |
| **Woo** | Conecta-se facilmente com novas pessoas | Pode ser superficial nas relações de longo prazo | Excelente para stakeholder management |

---

### Relationship Building — Constroem times coesos

| Força | Essência | Risco de excesso | Implicação para PM |
|---|---|---|---|
| **Adaptability** | Flui com o presente, responde bem a mudanças | Pode parecer sem direção ou inconsistente | Excelente para contextos ágeis — trabalhar visão de longo prazo |
| **Connectedness** | Enxerga como tudo está interligado | Pode ser abstrato demais para stakeholders práticos | Bom para ver impactos sistêmicos de decisões |
| **Developer** | Encontra potencial nas pessoas e as desenvolve | Pode priorizar desenvolvimento sobre resultado | Excelente como PM que eleva o time — calibrar com entregas |
| **Empathy** | Sente o que os outros estão sentindo | Pode absorver emoções do time, gerando paralisia | Excelente para research e discovery com usuários |
| **Harmony** | Busca consenso e evita conflito | Pode adiar ou evitar decisões difíceis | Bom para alinhamento — calibrar para não evitar conflito produtivo |
| **Includer** | Garante que todos se sintam parte do grupo | Pode sacrificar velocidade por inclusão | Bom para times diversos — calibrar decisão por consenso |
| **Individualization** | Enxerga o que é único em cada pessoa | Pode ser difícil de escalar | Excelente para coaching e stakeholder management |
| **Positivity** | Energia que contagia o time | Pode minimizar problemas reais com otimismo | Excelente para moral do time — ensinar a nomear os problemas |
| **Relator** | Relacionamentos profundos com pessoas próximas | Pode ser percebido como exclusivo | Excelente para confiança com time core e stakeholders-chave |

---

### Strategic Thinking — Absorvem e analisam para decidir melhor

| Força | Essência | Risco de excesso | Implicação para PM |
|---|---|---|---|
| **Analytical** | Busca dados e evidências antes de concluir | Pode travar por falta de dados perfeitos | Excelente para cultura de dados — calibrar para decidir com imperfeição |
| **Context** | Entende o presente pelo passado | Pode ficar preso no histórico | Bom para evitar erros repetidos — calibrar para aceitar novidade |
| **Futuristic** | Visão clara e inspiradora de futuro | Pode frustrar o time quando a visão está distante demais | Excelente para roadmap de longo prazo |
| **Ideation** | Conecta conceitos aparentemente não relacionados | Pode gerar muitas ideias sem executar nenhuma | Excelente para brainstorming — calibrar com critérios de priorização |
| **Input** | Coleta e guarda informações relevantes | Pode acumular sem sintetizar ou agir | Bom para competitive intelligence — trabalhar síntese e uso |
| **Intellection** | Pensa profundamente antes de agir | Pode ser percebido como lento ou distante | Excelente para problemas complexos — trabalhar comunicação do processo |
| **Learner** | Energizado pelo processo de aprender coisas novas | Pode começar coisas novas antes de terminar as anteriores | Excelente para adaptação — calibrar foco e entrega |
| **Strategic** | Identifica o melhor caminho em meio à complexidade | Pode parecer que pula passos para os outros | Excelente para liderança de produto — calibrar comunicação do raciocínio |

## Cruzamentos Relevantes para PM

| Combinação | Implicação |
|---|---|
| Activator + Analytical | Tensão interna: quer agir mas precisa de dados. Pode ser bloqueante ou gerar arrependimento |
| Achiever + Learner | Energia enorme para crescer — risco de burnout se não houver gestão intencional |
| Strategic + Command | Perfil natural de liderança de produto sênior |
| Harmony + Significance | Quer impacto mas evita conflito — pode comprometer decisões difíceis |
| Empathy + Deliberative | Excelente em research; pode ser lento em decisão quando os dados são ambíguos |
| Futuristic + Activator | Visão + energia para executar — calibrar execução estruturada para não virar caos |
| Developer + Responsibility | PM que cuida do time e das entregas — risco de assumir demais sem delegar |

## Como Usar os Top 5 no Coaching

1. Identificar onde cada força está sendo bem aplicada (confirmar e amplificar)
2. Identificar onde uma força em excesso está gerando problema (nomear o padrão sem julgamento)
3. Identificar gaps que são difíceis de fechar só com força de vontade — o que pode compensar
4. Usar os top 5 para escolher projetos e desafios de desenvolvimento que ativam as forças naturais
